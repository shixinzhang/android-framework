# sx-framework
Some base function classes for android development. 

从零开始创建自己的 Android 框架，主要包括：

- network
 - 第三方网络库使用、分析
 - 自己设计、封装 OkHttp
- image
 - 第三方图片加载库使用、分析
 - 自己设计、封装图片加载库
- cache
 - 三级缓存的设计与分析
- hybrid
 - WebView 常见功能封装
 - Weex 使用与分析
 - 参考 Weex 创建 hybrid 组件
- utils
 - 常见工具类
- views
 - 常见布局
